# CT-Eye: Citizen Safety Reporting Platform

CT-Eye was built for NaijaHacks 2019 as a citizen-safety platform for reporting, visualizing, and analyzing crime-related information in Nigeria. The project focuses on helping citizens report incidents more safely while giving the public and relevant authorities access to better crime intelligence.

## Problem

Many crimes and safety incidents go unreported because victims and witnesses may fear exposure, stigma, retaliation, or the friction of reporting through traditional channels. This makes it harder for citizens, support organizations, and law enforcement agencies to understand where incidents are happening and how risks are changing over time.

The original use case focused on Lagos, with Twitter/social data considered as one source of safety signals.

## Proposed Solution

CT-Eye proposes an independent citizen-centered reporting platform where users can submit and access non-sensitive crime information through simple digital channels.

The platform is designed around three goals:

1. Make incident reporting easier and safer for citizens.
2. Provide a structured repository of crime and safety data.
3. Visualize safety patterns so citizens and agencies can understand high-risk areas.

## Core Features

### View Danger Zones

Displays possible danger spots using geospatial analysis and mapped incident clusters.

### Generate Report

Guides users through an interactive chatbot flow for submitting an incident report. This is intended to reduce the burden on users who may not be ready to write a formal report after a stressful event.

### Send WhatsApp Report

Allows a user to send a direct report through WhatsApp to relevant contacts or emergency channels.
